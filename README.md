# Learn Beyond Beginner Python

A collection of Python learning materials and tutorials covering intermediate and advanced Python concepts.

## Project Structure

This repository contains several educational modules exploring key Python concepts:

### Core Files

- **`python-typing.py`** - Type hints and static typing in Python
- **`higher-order-functions.py`** - Functions that operate on other functions
- **`recursion.py`** - Recursive function patterns and techniques

### Object-Oriented Programming

- **`classes_tutorial.py`** - Comprehensive guide to Python classes
- **`dataclass_tutorial.py`** - Introduction to dataclasses with examples
- **`class_v_dataclass.py`** - Comparison between traditional classes and dataclasses

### Utilities

- **`arg_checker.py`** - Argument validation utilities
- **`main.py`** - Main entry point and examples

## Topics Covered

- **Type Hints** - Modern Python typing syntax and best practices
- **Higher-Order Functions** - Function composition and functional programming patterns
- **Recursion** - Recursive algorithms and techniques
- **Classes** - Object-oriented programming fundamentals
- **Dataclasses** - Simplified class definitions with `@dataclass` decorator
- **Argument Checking** - Input validation patterns

## Getting Started

1. Explore individual files to learn specific concepts
2. Run examples using Python 3.7+:
   ```bash
   python filename.py
   ```

## Usage

Each file can typically be run independently as a learning module:

```bash
python classes_tutorial.py
python python-typing.py
python higher-order-functions.py
```

*Happy learning! This repository is designed for practice and exploration of Python concepts beyond the beginner level.*
